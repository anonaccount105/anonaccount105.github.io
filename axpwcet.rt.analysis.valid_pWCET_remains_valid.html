We plan to replace every single job cost using the described procedure repeated for each job. However, note that the assumption axiomatic_pWCET_pWCET (Œº_of S) is stated only for the initial system S. After we apply the transformation, we create a new system S' := replace_job_pET j S; hence, in order to be able to use axiomatic_pWCET again, we must ensure that it holds in the system S'.
